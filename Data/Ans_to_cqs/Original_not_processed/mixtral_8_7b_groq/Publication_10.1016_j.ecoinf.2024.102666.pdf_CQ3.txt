2. Instance segmentation: This is a more advanced annotation technique, where each instance of an object is segmented and labeled in an image. This technique provides pixel-level accuracy and is useful for tasks that require detailed understanding of object shapes and positions, such as semantic segmentation and object counting.

3. Pixel-level annotation: Similar to instance segmentation, pixel-level annotation involves labeling each pixel in an image according to the object it belongs to. This technique is useful for tasks that require detailed understanding of the image content, such as material classification and scene understanding.

4. Semi-supervised and weakly-supervised learning: These are techniques that use a combination of labeled and unlabeled data for training deep learning models. Semi-supervised learning uses a small amount of labeled data along with a large amount of unlabeled data, while weakly-supervised learning uses weakly-labeled data, such as image-level labels or bounding boxes, instead of pixel-level annotations. These techniques can reduce the amount of manual annotation required and improve the efficiency of the annotation process.
