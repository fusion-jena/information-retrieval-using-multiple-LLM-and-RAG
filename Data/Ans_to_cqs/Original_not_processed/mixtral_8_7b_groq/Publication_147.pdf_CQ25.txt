The deep learning model in question is a segmentation convolutional neural network (CNN), specifically a U-Net architecture. This model is used for land cover classification, with a focus on wetland class mapping in the Boreal Forest region of Alberta, Canada. The CNN is implemented in the Python programming language using the Keras library and takes various inputs such as ARI, Band 2, Band 3, Band 4, DEM, NDVI, NDWI, PSRI, REIP, TPI, TRI, TWI, VBF, and VH. These inputs are used to optimize the deep learning model for wetland class mapping. The purpose of this deep learning model is to generate a wetland inventory style product by classifying land cover types in large-scale remote sensing data acquired from Sentinel-1, Sentinel-2, and ALOS DEM data in Google Earth Engine. The accuracy of the CNN-generated product is then tested against three validation data sets, including two photo-interpreted and one field data set.
