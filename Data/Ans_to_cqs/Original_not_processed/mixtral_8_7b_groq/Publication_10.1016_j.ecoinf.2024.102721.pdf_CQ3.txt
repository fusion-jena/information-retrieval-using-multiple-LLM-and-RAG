To provide a comprehensive answer, I will briefly explain some common data annotation techniques used in deep learning, especially for computer vision tasks:

1. Image-level annotation: This is the simplest form of annotation where the image is assigned a single label, denoting the object class or scene category present in the image.
2. Bounding box annotation: In this technique, a rectangular box is drawn around the object of interest in the image. This helps the model to learn the spatial location and size of the object.
3. Semantic segmentation: Here, each pixel in the image is assigned a class label, indicating the object class it belongs to. This helps the model to learn the precise object boundaries and shapes.
4. Instance segmentation: This is an extension of semantic segmentation, where each object instance is separated and assigned a unique label. This helps the model to differentiate between individual objects of the same class.
5. Panoptic segmentation: It combines semantic and instance segmentation, where each pixel is assigned a class label and instances are separated. It is useful for understanding the scene context and relationships between objects.
6. Keypoint annotation: In this technique, specific points or landmarks on the object are annotated, such as joints in human pose estimation.

These annotation techniques enable deep learning models to learn various aspects of visual data, such as object location, shape, class, and relationships, leading to better performance in computer vision tasks.
