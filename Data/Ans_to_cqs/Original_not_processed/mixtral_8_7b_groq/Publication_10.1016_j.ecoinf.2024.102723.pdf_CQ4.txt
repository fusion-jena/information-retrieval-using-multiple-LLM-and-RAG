Some common data augmentation techniques in deep learning include flipping, rotating, and scaling.

Flipping is a technique that involves flipping an image horizontally or vertically to create a new training example. This can help the model learn to recognize features that are invariant to orientation.

Rotating involves rotating an image by a certain angle to create a new training example. This can help the model learn to recognize features that are invariant to rotation.

Scaling involves changing the size of an image to create a new training example. This can help the model learn to recognize features that are invariant to scale.

Other data augmentation techniques include brightness and contrast adjustments, adding noise, and cropping.
