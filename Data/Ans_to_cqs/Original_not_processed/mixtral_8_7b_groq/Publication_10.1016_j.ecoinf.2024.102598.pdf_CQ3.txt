Some common data annotation techniques used in deep learning include:

1. Bounding box annotation: This technique involves drawing rectangular boxes around objects of interest in an image. It helps the model to identify and locate objects within the image.

2. Instance segmentation: In this method, each instance of an object is segmented and labeled in the image. It provides a more detailed and precise annotation compared to bounding box annotation.

3. Semantic segmentation: This technique involves dividing an image into multiple segments and labeling each segment based on the object or scene it represents. It is useful for understanding the overall context and layout of an image.

4. Pixel-level annotation: This method involves labeling each pixel in an image based on the object or scene it represents. It is a highly detailed and time-consuming process but can provide very accurate results.

5. Scene text annotation: This technique involves identifying and extracting text from images. It is useful for applications like image-based OCR and text recognition.
