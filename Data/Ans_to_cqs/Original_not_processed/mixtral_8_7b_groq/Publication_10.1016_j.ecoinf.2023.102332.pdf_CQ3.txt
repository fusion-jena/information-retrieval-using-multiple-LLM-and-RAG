Word2vec is a word embedding technique that represents words as high-dimensional vectors, capturing semantic relationships between them.

Data annotation techniques are crucial for training deep learning models, especially for tasks such as object detection, semantic segmentation, and instance segmentation. These techniques involve labeling data with relevant information, such as bounding boxes, pixel-level masks, or class labels.

Bounding box annotation is a technique used for object detection tasks, where a bounding box is drawn around an object of interest in an image. This technique is used to train models to detect and classify objects within an image.

Instance segmentation is a more advanced technique used for semantic segmentation tasks, where each pixel in an image is labeled with the class of the object it belongs to, and each instance of the object is distinguished from others. This technique is used to train models to identify and segment individual objects within an image.
