Image-level labels are the most basic form of annotation, where each image is assigned a single label indicating the object class present in the image. For example, in the context of plant species recognition, each image could be labeled with the species of the plant it contains.

Bounding box annotations are used to localize objects within an image. Each object is enclosed in a rectangular box, and the box coordinates are used as part of the training data. This type of annotation is useful for tasks such as object detection, where the goal is to identify and locate objects within an image.

Segmentation masks provide the most detailed level of annotation, where each pixel in the image is assigned a label indicating the object class it belongs to. This type of annotation is useful for tasks such as semantic segmentation, where the goal is to classify each pixel in the image.
