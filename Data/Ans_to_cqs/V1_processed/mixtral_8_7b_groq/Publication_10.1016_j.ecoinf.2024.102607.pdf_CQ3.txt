In deep learning, data annotation is a crucial step where experts manually label or annotate data, enabling models to learn and make predictions about new, unseen data. There are several data annotation techniques used in the deep learning pipeline, including:

1. **Image tagging or classification:** This is the simplest form of annotation where an expert assigns a single label to an entire image. For example, an image may be labeled as 'forest', 'urban', or 'water'.

2. **Bounding box annotation:** In this technique, a rectangular box is drawn around an object of interest in an image. This helps the model to focus on the object and learn its features while ignoring the background. Bounding box annotation is commonly used in object detection tasks.

3. **Semantic segmentation:** This technique involves labeling each pixel in an image with a specific class. Semantic segmentation helps the model to understand the spatial distribution of objects within an image.

4. **Instance segmentation:** Instance segmentation is an extension of semantic segmentation, where each instance of an object is segmented and assigned a unique identifier. This technique is useful for counting and tracking objects within an image.

5. **Panoptic segmentation:** Panoptic segmentation combines semantic and instance segmentation, providing a complete understanding of the scene. It labels each pixel in an image with a class and assigns a unique identifier to each instance of an object.
