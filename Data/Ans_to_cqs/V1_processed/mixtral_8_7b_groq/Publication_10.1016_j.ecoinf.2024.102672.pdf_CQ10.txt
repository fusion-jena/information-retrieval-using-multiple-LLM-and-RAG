The code repository link for the deep learning pipeline can be found on GitHub. The researchers have made the pre-processed dataset and the Python code used in the study available for facilitating the implementation of this framework. The link to the GitHub repository is: <https://github.com/IFCA-Advanced-Computing/Thermocline_prediction>. Additionally, the code has been assigned a Digital Object Identifier (DOI) for easy citation and access: doi:<https://doi.org/10.5281/zenodo.11479475>. With this information, anyone interested in the deep learning pipeline used in the study can access and use the code.
