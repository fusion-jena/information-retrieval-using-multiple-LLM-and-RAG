Data annotation is the process of labeling or marking up data so that it can be used in machine learning models. In the context of computer vision, this typically involves annotating images or videos. Here are some common data annotation techniques:

1. Image classification: This is the simplest form of data annotation, where an image is labeled with a single class or category. For example, an image might be labeled as "dog" or "cat".
2. Bounding box annotation: In this technique, a bounding box is drawn around an object of interest in an image. This allows the model to learn the spatial location and extent of the object. Bounding box annotation is commonly used for object detection tasks.
3. Semantic segmentation: This is a more advanced form of data annotation, where each pixel in an image is labeled with a class or category. This allows the model to learn the precise shape and location of objects in an image. Semantic segmentation is commonly used for scene understanding tasks.
4. Instance segmentation: This is a further refinement of semantic segmentation, where each instance of an object is labeled separately. This allows the model to distinguish between multiple instances of the same object in an image. Instance segmentation is commonly used for object detection and tracking tasks.
5. Keypoint annotation: In this technique, specific keypoints or landmarks are identified and labeled in an image. This is commonly used for tasks such as facial recognition or human pose estimation.

These are just a few examples of data annotation techniques that are commonly used in the deep learning pipeline. The specific technique used will depend on the task at hand and the nature of the data being analyzed.
