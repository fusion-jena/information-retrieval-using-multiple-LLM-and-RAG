Data annotation is the process of labeling data so that it can be used to train machine learning models. In the context of computer vision, data annotation involves labeling images or video frames with objects of interest. Here are some common data annotation techniques used in deep learning:

1. Bounding box annotation: This technique involves drawing a bounding box around an object of interest in an image. The bounding box contains the coordinates of the object, which can be used to train object detection models.
2. Image segmentation: This technique involves dividing an image into multiple regions or segments, where each segment represents a different object or part of an object. Image segmentation can be used for tasks such as object detection, semantic segmentation, and instance segmentation.
3. Image tagging: This technique involves labeling an image with one or more tags that describe the content of the image. Image tagging can be used for tasks such as image classification and retrieval.
4. Keypoint annotation: This technique involves identifying and labeling key points or landmarks on an object of interest in an image. Keypoint annotation can be used for tasks such as object detection, pose estimation, and facial recognition.
5. Text annotation: This technique involves labeling text data with entities, sentiments, or other relevant information. Text annotation can be used for tasks such as text classification, sentiment analysis, and named entity recognition.
