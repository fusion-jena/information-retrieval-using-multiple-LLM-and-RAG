In the field of remote sensing, data annotation techniques are often used to train deep learning models for image classification and object detection tasks. Some common data annotation techniques used in deep learning pipelines include:

1. Bounding box annotation: This technique involves drawing a bounding box around an object of interest in an image. The bounding box serves as a label for the object and is used to train deep learning models for object detection tasks.
2. Instance segmentation: This technique involves segmenting individual objects in an image and labeling each pixel within the object. Instance segmentation is a more fine-grained approach than bounding box annotation and is often used for tasks that require precise object localization and segmentation.
3. Semantic segmentation: This technique involves segmenting an image into different regions or classes based on their semantic meaning. Semantic segmentation is a less fine-grained approach than instance segmentation and is often used for tasks that require classifying regions of an image.
