2. **Semantic segmentation:** This method assigns a class label to each pixel in an image, creating a pixel-wise classification map. Semantic segmentation is useful for understanding the spatial extent of objects and their context within an image. For example, it can be used to differentiate between different types of vegetation or habitats in a satellite image.
3. **Instance segmentation:** This technique combines object detection and semantic segmentation, assigning a unique identifier to each instance of an object in an image. Instance segmentation allows for counting and tracking individual objects, which can be helpful in ecological studies for monitoring populations or tracking individual animals.
