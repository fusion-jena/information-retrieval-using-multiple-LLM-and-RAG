2. Summarization: Model summaries can provide an overview of the fitted model, including coefficients, standard errors, and statistical significance for each predictor. This information can help assess the importance of individual predictors and their contribution to the model.

3. Validation: Model validation is an essential step in SDM evaluation. Various metrics can be calculated to assess the model's predictive performance, such as the Area Under the Curve (AUC) of the Receiver Operating Characteristic (ROC) curve, True Skill Statistic (TSS), or Cohen's Kappa. These metrics can help determine the model's ability to discriminate between presences and absences or predict occurrence probabilities accurately.

4. Confusion matrix: A confusion matrix is a table that summarizes the performance of a binary classification model, such as a presence-absence model. It compares the predicted values with the observed values and provides information on false positives, false negatives, true positives, and true negatives. From the confusion matrix, additional metrics like precision, recall, and F1-score can be calculated.

5. Saliency maps: Saliency maps are a technique for visualizing the importance of input features for a specific model prediction. They can help identify the most relevant predictors for a given location or species occurrence probability.
