In general, data annotation techniques for deep learning tasks include:

1. Image Classification: This involves assigning one label to an entire image. It is useful when identifying objects that occupy most of the image space.

2. Object Detection: This technique identifies multiple objects within an image and draws bounding boxes around them. Each bounding box is assigned a corresponding label.

3. Semantic Segmentation: This method labels every pixel in an image according to its enclosing object category. It helps distinguish between overlapping objects and background elements.

4. Instance Segmentation: Similar to semantic segmentation, this technique also labels individual pixels. However, it distinguishes between instances of the same class, allowing for more precise identification of separate objects belonging to the same category.
