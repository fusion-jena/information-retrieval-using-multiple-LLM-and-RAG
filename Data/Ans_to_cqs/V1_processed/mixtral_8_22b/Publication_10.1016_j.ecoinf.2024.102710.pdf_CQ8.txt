The code repository of the deep learning pipeline is available on GitHub. Specifically, it can be found at <https://github.com/HKathman/pam_annotation_experiments>. Additionally, other relevant repositories include <https://github.com/kahst/BirdNET-Analyzer/tree/main/checkpoints/V2.4>, which contains checkpoints for the BirdNET-Analyzer V2.4, and <https://tfhub.dev/google/vggish/1> and <https://tfhub.dev/google/yamnet/1>, which provide TensorFlow Hub modules for audio feature extraction using VGGish and YAMNet, respectively.
